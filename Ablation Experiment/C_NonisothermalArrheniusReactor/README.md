# NonisothermalArrheniusReactor 模型文档

## 1) 模型目的与范围

本模型为非等温批式化学反应器模型，旨在模拟反应物A发生不可逆一级放热反应生成产物B的动态过程。模型遵循阿伦尼乌斯动力学，并伴随冷却换热过程。其涵盖的物理机制包括：
- 阿伦尼乌斯动力学
- 化学计量关系与物质守恒
- 放热反应热效应
- 能量守恒与热衡算
- 对流换热冷却

## 2) 组件清单

本模型未使用 Modelica 标准库 (MSL) 组件，完全基于原生变量与方程构建，无外部组件依赖。

## 8) 运行与复现方法

1. **环境准备**：确保已安装支持 Modelica 的仿真环境（如 Dymola, OpenModelica 等）。
2. **获取模型文件**：将模型代码保存为相对路径下的文件 `./model/NonisothermalArrheniusReactor.mo`。
3. **加载模型**：在仿真环境中打开或导入 `./model/NonisothermalArrheniusReactor.mo` 文件。
4. **仿真设置**：配置仿真参数：终止时间设为 `100`，求解器选择 `dassl`，容差设为 `1e-06`。
5. **执行仿真**：运行仿真并查看目标变量（C_A, C_B, T 等）的结果曲线与数据。
