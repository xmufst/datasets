# NewtonCooling 模型文档

## 1) 模型目的与范围

本模型（`NewtonCooling`）旨在模拟一个具有集总参数的热质在恒定温度环境中的冷却过程。该模型基于牛顿冷却定律，通过能量守恒方程描述物体温度随时间的变化。模型包含的物理机制包括：集总热容、对流换热、温度指数衰减、能量守恒以及热时间常数推导。

## 2) 组件清单

本模型是一个自包含的 Modelica 模型，没有使用 Modelica 标准库（MSL）中的组件。所有方程均在模型中直接定义。

## 3) 参数表

模型包含以下常数参数：

| 参数名 | 值 | 单位 | 描述 |
| :--- | :--- | :--- | :--- |
| `m` | 1.0 | kg | 物体质量 |
| `cp` | 500.0 | J/(kg·K) | 比热容 |
| `hA` | 100.0 | W/K | 综合换热系数 |
| `T0` | 100.0 | °C | 初始物体温度 |
| `T_env` | 20.0 | °C | 环境温度 |

## 4) 初始条件

模型在仿真开始时 (`time = 0`) 为以下变量设置了初始值：

| 变量 | 初始值 | 单位 | 描述 |
| :--- | :--- | :--- | :--- |
| `T` | 100.0 | °C | 初始物体温度 |
| `E_dissipated` | 0 | J | 初始累积散热量 |

## 5) 控制逻辑说明

此模型不包含任何控制逻辑。系统的动态完全由基于牛顿冷却定律和能量守恒的微分方程描述。

## 6) 仿真设置

本模型的仿真使用以下配置运行：
*   **模型文件路径**: `./model/NewtonCooling.mo`
*   **仿真时长**: 25.0 秒
*   **积分方法**: `dassl`
*   **容差**: 1e-06

## 7) 结果概览

下表列出了关键目标变量在整个仿真过程中的统计摘要。模型输出符合预期，物体温度`T`从初始的 100°C 指数衰减至接近环境温度 20°C。

| 变量 | 开始值 | 最终值 | 最小值 | 最大值 | 单位 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| `T` | 100.000000 | 20.539419 | 20.539419 | 100.000000 | °C |
| `dT` | 80.000000 | 0.539419 | 0.539419 | 80.000000 | K |
| `Q_dot` | 8000.000000 | 53.941854 | 53.941854 | 8000.000000 | W |
| `E_thermal` | 40000.000000 | 269.709272 | 269.709272 | 40000.000000 | J |
| `E_dissipated` | 0.000000 | 39730.290728 | 0.000000 | 39730.290728 | J |
| `tau` | 5.000000 | 5.000000 | 5.000000 | 5.000000 | s |

> **热时间常数** `tau` = m * cp / hA = 5.0 秒，与仿真中观察到的指数衰减特征相符。

## 8) 运行与复现方法

1.  **准备环境**：确保已安装支持 Modelica 的仿真环境（如 Dymola、OpenModelica）。
2.  **加载模型**：在仿真工具中打开或加载位于 `./model/NewtonCooling.mo` 的模型文件。
3.  **设置仿真参数**：
    *   仿真时长：25.0 秒。
    *   积分方法：`dassl`。
4.  **运行仿真**：执行仿真。
5.  **使用脚本（可选）**：对于 OpenModelica，可创建并运行如下 `.mos` 脚本：
    ```modelica
    // run_NewtonCooling.mos
    loadFile("./model/NewtonCooling.mo");
    simulate(NewtonCooling, stopTime=25.0, method="dassl", tolerance=1e-6);
    plot({T, T_env});
    ```
    执行命令：`omc run_NewtonCooling.mos`
