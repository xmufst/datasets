# SpringMassDamper 模型文档

## 1) 模型目的与范围
本模型旨在模拟单自由度弹簧-质量-阻尼器系统的自由振动衰减过程。通过建立包含惯性、刚度与阻尼效应的动力学方程，展示系统在给定初始位移下的动态响应及能量耗散规律。
**物理机制涵盖**：
- 线性弹簧恢复力
- 粘性阻尼耗散
- 自由振动衰减
- 机械能耗散

## 2) 组件清单
本模型未使用 Modelica 标准库 (MSL) 的物理组件（如 Mass, Spring 等），而是采用底层数学方程与基础变量类型进行搭建：
- **参数组件**：`parameter Real`（用于定义质量、刚度、阻尼及初始条件）
- **变量组件**：`Real`（用于定义位移、速度、加速度、力及能量等状态与代数变量）

## 3) 参数表
| Name | Value | Unit | Description |
| :--- | :--- | :--- | :--- |
| `m` | 1.0 | kg | 质量块质量 |
| `k` | 100.0 | N/m | 弹簧刚度 |
| `c` | 2.0 | N·s/m | 阻尼系数 |
| `x0` | 0.5 | m | 初始位移 |
| `v0` | 0.0 | m/s | 初始速度 |

## 4) 初始条件
| Variable | Value | Description |
| :--- | :--- | :--- |
| `x` | `x0` (0.5 m) | 质量块初始位移 |
| `v` | `v0` (0.0 m/s) | 质量块初始速度 |

## 5) 控制逻辑说明
本模型无控制逻辑。系统为纯物理自由振动系统，响应完全由初始条件和系统固有属性（质量、刚度、阻尼）决定。

## 8) 运行与复现方法
1. **准备模型文件**：确保模型文件位于相对路径 `./model/SpringMassDamper.mo`。
2. **运行环境**：使用支持 Modelica 的仿真环境（如 OpenModelica、Dymola 等）。
3. **加载与仿真**（以 OpenModelica 为例）：
   ```modelica
   loadFile("./model/SpringMassDamper.mo");
   simulate(SpringMassDamper, stopTime=10.0, stepSize=0.01, method="dassl", tolerance=1e-06);
   ```
4. **结果绘图**：
