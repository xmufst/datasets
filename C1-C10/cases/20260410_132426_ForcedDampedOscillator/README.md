# ForcedDampedOscillator 模型文档

## 1) 模型目的与范围
**模型名称**：ForcedDampedOscillator  
**模型描述**：受外部正弦激振力作用的弹簧-质量-阻尼器系统的动态响应及能量功率分析模型。  
**物理机制**：涵盖受迫振动、阻尼振动、能量耗散以及机械能转换过程。本模型旨在研究系统在外部持续激励下的瞬态与稳态响应，以及在此过程中的能量流动与耗散规律。

## 2) 组件清单
本模型未使用 Modelica 标准库 (MSL) 中的物理组件（如弹簧、阻尼器等组件），而是采用原生 Modelica 变量和微分方程进行底层物理建模。使用的变量类型包括：
- `parameter Real`：物理参数与初始条件参数
- `final parameter Real`：导出参数（固有频率、阻尼比等）
- `Real`：状态变量及目标观测变量

## 5) 控制逻辑说明
本模型**不包含控制逻辑**。系统为纯物理动力学开环模型，外部激励遵循给定的正弦函数规律 `F_ext = F0 * sin(omega * time)`，运动方程依据牛顿第二定律 `m * a = F_ext - F_spring - F_damper` 演化。

## 6) 仿真设置

| :--- | :--- |
| 停止时间 | 10 s |
| 步长 | 0.01 s |
| 求解器 | dassl |
| 容差 | 1e-06 |

## 8) 运行与复现方法
1. **环境准备**：安装支持 Modelica 的仿真环境（如 OpenModelica、Dymola 等）。
2. **获取模型文件**：模型文件位于 `./model/ForcedDampedOscillator.mo`。
3. **加载模型**：在仿真环境中打开或导入上述 `.mo` 文件。
4. **仿真配置**：在仿真设置界面，将停止时间设为 `10`，通信步长设为 `0.01`，求解器选择 `dassl`，容差设为 `1e-06`。
5. **执行仿真**：运行仿真并检查结果，重点观察 `x`、`v`、`E_total` 及 `P_dissipated` 等目标变量曲线，以验证受迫阻尼振动的能量耗散与动态响应特性。
