# SimplePendulum 模型文档

## 1) 模型目的与范围
**目的**：模拟理想刚性单摆在均匀重力场中的自由摆动过程。
**范围**：采用完整非线性运动方程（未对摆角进行小角度近似），验证保守系统的能量守恒特性，并分析摆角、角速度及摆锤空间位置随时间的变化规律。涵盖的物理机制包括：
- 刚体旋转运动
- 均匀重力场作用
- 保守系统能量守恒

## 2) 组件清单
本模型未使用 Modelica 标准库 (MSL) 组件，完全基于基础 `Real` 变量和微分代数方程构建，属于纯数学解析建模。

## 3) 参数表
| Name | Value | Unit | Description |
| :--- | :--- | :--- | :--- |
| `m` | 1.0 | kg | 摆锤质量 |
| `g` | 9.81 | m/s2 | 重力加速度 |
| `L` | 1.0 | m | 摆长 |
| `theta0` | 0.5 | rad | 初始摆角 |
| `omega0` | 0.0 | rad/s | 初始角速度 |

## 4) 初始条件
| Variable | Value | Description |
| :--- | :--- | :--- |
| `theta` | `theta0` (0.5 rad) | 初始摆角 |
| `omega` | `omega0` (0.0 rad/s) | 初始角速度 |

## 5) 控制逻辑说明
本模型不包含任何控制逻辑，为纯物理动力学开环模型。

## 6) 仿真设置
| Setting | Value |
| :--- | :--- |
| 停止时间 | 10 s |
| 步长 | 0.01 s |
| 求解器 | dassl |
| 容差 | 1e-06 |

## 8) 运行与复现方法
1. 确保已安装支持 Modelica 的仿真环境（如 OpenModelica、Dymola 等）。
2. 将模型文件放置于相对路径 `./model/SimplePendulum.mo`。
3. 在 OpenModelica 环境下，可通过以下脚本命令加载模型并复现仿真结果：

```modelica
// 加载模型文件
loadFile("./model/SimplePendulum.mo");

// 执行仿真
simulate(SimplePendulum, stopTime=10.0, stepSize=0.01, method="dassl", tolerance=1e-06);
