# FallingWithDrag 模型文档

## 1. 模型目的与范围
本模型 `FallingWithDrag` 旨在模拟物体在重力作用下自由下落，并受到与速度平方成正比的空气阻力的物理过程。它是一个基于物理方程的动态系统模型，可用于分析下落物体的速度、加速度、受力及能量转换等关键物理量随时间的变化，并特别关注其趋近于终端速度的动态过程。

## 2. 组件清单
此模型基于基本方程构建，未直接使用 Modelica 标准库 (MSL) 中的预定义组件。模型的核心由一个常微分方程系统构成，用于计算运动学和动力学变量。

## 3. 参数表
模型包含以下可调整参数：

| 名称 | 值 | 单位 | 描述 |
| :--- | :--- | :--- | :--- |
| `m` | 1.0 | kg | 物体质量 |
| `g` | 9.81 | m/s² | 重力加速度 |
| `b` | 0.04 | kg/m | 综合阻力系数（用于计算阻力 F_drag = b \* v \* abs(v)） |
| `v0` | 0.0 | m/s | 初始下落速度 |

## 4. 初始条件
仿真开始时，以下变量的值被设定为：

| 变量 | 值 | 描述 |
| :--- | :--- | :--- |
| `y` (下落距离) | 0 | 初始下落距离 |
| `v` (速度) | 0.0 m/s | 初始下落速度（与参数 `v0` 一致） |

## 5. 控制逻辑说明
本模型为纯粹的物理仿真模型，不包含任何外部或离散的控制逻辑。

## 6. 仿真设置
用于生成参考结果的仿真配置如下：

| 设置项 | 值 |
| :--- | :--- |
| 仿真停止时间 | 10 s |
| 输出/通信步长 | 0.01 s |
| 求解器方法 | dassl |
| 容差 | 1e-06 |

## 7. 结果概览
在默认参数和上述仿真设置下，模型计算得到的主要目标变量统计摘要如下：

| `v` | m/s | 0.000 | 8.701 | 0.000 | 8.701 | 速度持续增加，趋近终端速度 |

## 8. 运行与复现方法
1.  **加载模型**：在您的 Modelica 仿真环境（如 Dymola, OpenModelica）中，打开位于 `./model/FallingWithDrag.mo` 的模型文件。
2.  **设置仿真**：
    *   确保模型参数为第3节中列出的默认值。
    *   在仿真设置中，配置：
        *   停止时间 (Stop Time): `10`
        *   输出间隔/通信步长 (Output Interval/Number of Intervals): `0.01` (或等效设置)
        *   求解器 (Solver): `dassl`
        *   容差 (Tolerance): `1e-06`
3.  **执行仿真**：运行瞬态仿真。
4.  **查看结果**：仿真结束后，您将得到与第7节概览一致的结果。可以绘制变量（如 `v`, `v_ratio`, `F_net`）随时间变化的曲线以进行详细分析。
