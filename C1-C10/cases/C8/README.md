# GravityTank 模型文档

## 1. 模型目的与范围

本模型 `GravityTank` 模拟一个带有底部孔口的垂直圆柱形储罐。其主要物理机制是液体在重力作用下通过孔口排出。模型可用于研究液位高度、流出速率等变量随时间的变化过程，例如分析储罐排空时间或评估不同孔口尺寸对排水性能的影响。

## 2. 组件清单 (MSL)

本模型主要基于 Modelica 标准库 (MSL) 的基本类型和单位定义构建，未直接实例化复杂的预定义物理组件。所使用的 MSL 元素包括：
*   `Modelica.Units.SI`: 用于定义具有国际单位制 (SI) 的参数和变量。

## 3. 参数表

| 名称 | 值 | 单位 | 描述 |
| :--- | :--- | :--- | :--- |
| `A_tank` | 0.1 | m² | 储罐的横截面积 |
| `A_orifice` | 0.01 | m² | 孔口的面积 |
| `Cd` | 0.61 | – | 流量系数（无量纲） |
| `g` | 9.81 | m/s² | 重力加速度 |
| `h0` | 2.0 | m | 初始液位高度 |
| `Q_in` | 0.0 | m³/s | 恒定流入速率 |

## 4. 初始条件

| 变量 | 值 | 描述 |
| :--- | :--- | :--- |
| `h` | `h0` (2.0 m) | 液位高度 |

模型在初始化时包含一个断言 (`assert`)，确保初始液位 `h0` 非负。

## 5. 控制逻辑说明

本模型不包含传统的连续反馈控制逻辑。但实现了一个**离散事件**用于物理边界保护：
*   **事件**: `when h <= 0 then`
*   **动作**: `reinit(h, 0)`
*   **目的**: 当液位 `h` 降至零或以下时，将 `h` 重新初始化为 0，防止液位在模型方程计算中变为负值。

## 6. 仿真设置

本次仿真的关键设置如下：

*   **仿真时长 (`stop_time`)**: 10 s
*   **输出步长 (`step_size`)**: 0.01 s
*   **求解器 (`solver_method`)**: `dassl`
*   **容差 (`tolerance`)**: 1e-06

## 7. 结果概览

下表汇总了目标变量在 `0-10 s` 仿真区间内的关键统计信息。

| 变量 | 单位 | 初值 (t=0s) | 终值 (t=10s) | 最小值 | 最大值 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **`h`** (液位高度) | m | 2.000 | 0.004 | 0.004 | 2.000 |
| **`Q_out`** (流出速率) | m³/s | 0.0382 | 0.00171 | 0.00171 | 0.0382 |
| **`v_out`** (孔口流速) | m/s | 6.264 | 0.280 | 0.280 | 6.264 |
| **`V_liquid`** (液体体积) | m³ | 0.200 | 0.000400 | 0.000400 | 0.200 |
| **`Q_net`** (净流入速率) | m³/s | -0.0382 | -0.00171 | -0.0382 | -0.00171 |

**结果解读**: 在无流入 (`Q_in=0`) 的情况下，储罐通过底部孔口持续排水。液位 `h`、流出量 `Q_out`、孔口流速 `v_out` 和液体体积 `V_liquid` 均随时间单调递减。净流量 `Q_net` 始终为负（表示净流出），其绝对值随液位降低而减小。仿真结束时，储罐接近排空状态。

## 8. 运行与复现方法

1.  **加载模型**: 在您的 Modelica 仿真环境 (如 Dymola, OpenModelica) 中，打开或加载模型文件 `./model/GravityTank.mo`。
2.  **检查参数**: 确认模型参数（第3节）与您的需求一致。可在仿真前在参数对话框中修改。
3.  **设置仿真**: 配置仿真参数，建议使用文档第6节中的设置 (`stop_time=10`, `tolerance=1e-6`, 求解器 `dassl`)。
4.  **运行仿真**: 执行瞬态仿真。
5.  **查看结果**: 绘制第7节中列出的目标变量，以复现上述结果。液位 `h` 应呈现从 `h0` 开始的平滑递减曲线，最终趋近于零。
