  // Variables with start value and unit
  Real h(unit="m", start=0.5) "水箱液位";
  Real h_error(unit="m") "液位跟踪误差";
  Real Q_in(unit="m3/s") "实际入流量";
  Real Q_out(unit="m3/s") "实际出流量";
  Real u_pi "PI控制器原始输出信号";

  // Standard library components
  Modelica.Blocks.Sources.Constant h_ref_signal(k=h_ref) "设定值模块";
  Modelica.Blocks.Math.Feedback feedback "反馈模块";

  // PI controller using P and Integrator components
  Modelica.Blocks.Math.Gain P_gain(k=Kp) "比例增益";
  Modelica.Blocks.Continuous.Integrator I_integrator(
    k=1/Ti,
    y_start=0,
    initType=Modelica.Blocks.Types.Init.InitialState) "积分器";
  Modelica.Blocks.Math.Add add "P和I相加";

  Modelica.Blocks.Nonlinear.Limiter limiter(
    uMin=0,
    uMax=1) "限幅模块";
  Modelica.Blocks.Math.Product product "乘法模块";

initial equation

equation
  // Connect standard library components

  // PI controller connections
  connect(feedback.y, P_gain.u);
  connect(feedback.y, I_integrator.u);
  connect(P_gain.y, add.u1);
  connect(I_integrator.y, add.u2);

  // Continue with limiter and product
  connect(add.y, limiter.u);
  connect(limiter.y, product.u1);
  connect(Qmax_constant.y, product.u2);

  // Connect tank level to feedback
  h = feedback.u2;

  // Assign outputs to variables for monitoring
  h_error = feedback.y;
  u_pi = add.y;
  Q_in = product.y;

  // Physical tank model (non-standard part)
  A_tank * der(h) = Q_in - Q_out;
  Q_out = k_out * sqrt(max(h, 0));
end TankLevelPI;

# TankLevelPI - 水箱液位 PI 控制系统文档

## 1. 模型目的与范围

本模型 `TankLevelPI` 模拟一个单水箱液位闭环控制系统。系统的控制目标是维持水箱液位 `h` 在预设的参考液位 `h_ref`。流入水箱的流量 `Q_in` 由一个比例积分（PI）控制器调节，以补偿由重力驱动的流出流量 `Q_out`（与液位的平方根成正比）。该系统体现了典型的工业过程控制逻辑，结合了基于质量守恒的物理模型和标准的反馈控制策略。

## 2. 组件清单

模型使用了以下来自 Modelica 标准库 (MSL) 的组件：

| 组件类型 (包路径) | 实例名 | 主要功能 |
| :--- | :--- | :--- |
| `Modelica.Blocks.Sources.Constant` | `h_ref_signal` | 产生恒定的液位设定点信号。 |
| `Modelica.Blocks.Math.Feedback` | `feedback` | 计算设定点与测量值之间的误差 (`h_ref - h`)。 |
| `Modelica.Blocks.Continuous.PI` | `PI_controller` | 执行比例-积分控制算法。 |
| `Modelica.Blocks.Nonlinear.Limiter` | `limiter` | 将控制器输出限制在 [0, 1] 的有效范围内。 |
| `Modelica.Blocks.Math.Product` | `product` | 将限幅后的控制信号与最大流入率相乘，得到实际流入率。 |
| `Modelica.Blocks.Sources.Constant` | `Qmax_constant` | 提供恒定的最大流入率参数。 |
| `Modelica.Blocks.Sources.RealExpression` | `level_signal` | 将模型变量 `h` 作为测量信号反馈给控制系统。 |

## 5. 控制逻辑说明

控制系统采用标准单回路 PI 反馈结构：
1.  **误差计算**：通过 `feedback` 组件实时计算液位设定值 `h_ref` 与实测液位 `h` 的误差 `h_error`。
2.  **PI 控制**：误差信号被送入 `PI_controller`。控制器输出 `u_pi` 包含比例和积分作用，用于消除稳态误差。
3.  **输出限幅**：`u_pi` 经过 `limiter` 被限制在 0 到 1 之间，代表从关闭 (0) 到全开 (1) 的阀门开度信号。
4.  **流量转换**：限幅后的开度信号与 `Q_in_max` 相乘，得到施加于物理水箱的实际入口流量 `Q_in`。
5.  **物理模型反馈**：水箱根据流入 `Q_in` 和流出 `Q_out` 的差值动态改变液位 `h`，`h` 通过 `level_signal` 反馈回控制系统，形成闭环。

**初始化**：PI 控制器积分状态的初始值通过参数 `y_start=0` 明确设置。
