# BoilerFeedwaterRatio 模型文档

## 1) 模型目的与范围
本模型 `BoilerFeedwaterRatio` 实现了一个锅炉给水比例控制系统。其核心目的是通过一个比例-积分（PI）控制器，调节给水阀门的开度，使得给水质量流量 (`W_feedwater`) 与蒸汽质量流量 (`W_steam`) 的比值稳定在设定值 (`R_set`)。模型包含了简化的汽包水位 (`h_drum`) 质量平衡动力学，用于模拟在蒸汽流量发生阶跃扰动时，控制系统维持水汽比例并稳定水位的过程。该模型适用于控制逻辑验证和动态特性分析。

## 2) 组件清单（MSL 组件）
模型使用了以下 Modelica 标准库 (MSL) 中的组件：
*   `Modelica.Blocks.Sources.Step`: 生成蒸汽流量的阶跃扰动。
*   `Modelica.Blocks.Sources.Constant`: 提供比例设定值 (`R_set`) 和最大给水流量 (`K_fw`)。
*   `Modelica.Blocks.Math.Division`: 计算实际的水汽比例 (`W_feedwater / W_steam`)。
*   `Modelica.Blocks.Math.Feedback`: 计算比例设定值与实际值之间的误差。
*   `Modelica.Blocks.Continuous.PI`: 实现比例-积分控制算法，根据误差输出控制信号。
*   `Modelica.Blocks.Nonlinear.Limiter`: 将控制器的输出限制在阀门有效开度范围 `[0, 1]` 内。
*   `Modelica.Blocks.Math.Product`: 将阀门开度与最大给水流量相乘，得到当前给水流量指令。

## 3) 参数表
| 参数名 | 值 | 单位 | 描述 |
| :--- | :--- | :--- | :--- |
| `A_eff` | 500.0 | kg/m | 等效汽包面积 (质量/长度) |
| `K_fw` | 3.0 | kg/s | 最大给水质量流量 |
| `W_steam_base` | 1.0 | kg/s | 蒸汽流量基值（扰动前） |
| `dW_steam` | 0.5 | kg/s | 蒸汽流量阶跃扰动量 |
| `t_dist` | 200.0 | s | 施加蒸汽流量扰动的时间 |
| `R_set` | 1.0 | 1 | 给水/蒸汽比例的设定点 |
| `Kp` | 3.0 | 1 | PI控制器的比例增益 |
| `Ti` | 10.0 | s | PI控制器的积分时间 |

## 4) 初始条件
*   **`h_drum` (汽包水位)** = `1.0 m`
*   **`PI_controller.integrator_state` (PI控制器积分状态)** = `0.0`

## 5) 控制逻辑说明
系统采用单回路反馈控制，目标是维持水汽比例 (`ratio_actual = W_feedwater / W_steam`) 恒定。
1.  **测量与计算**：实时测量给水流量 (`W_feedwater`) 和蒸汽流量 (`W_steam`)，通过除法器计算实际比例 (`ratio_actual`)。
2.  **误差生成**：通过反馈比较环节，计算实际比例与设定比例 (`R_set`) 的偏差 (`ratio_error`)。
3.  **控制律**：偏差信号送入 PI 控制器。控制器输出 (`u_pi`) 代表理论所需的阀门开度。
4.  **执行器限幅**：控制器输出经过限幅器，确保阀门开度 (`valve_opening`) 在物理可行的 `0` (全关) 到 `1` (全开) 之间。
5.  **流量计算**：最终的阀门开度与最大给水流量 (`K_fw`) 相乘，得到实际的给水流量指令，从而闭合回路。
6.  **被控对象**：给水与蒸汽流量之差，通过汽包的质量平衡方程 (`der(h_drum) = (W_feedwater - W_steam) / A_eff`)，影响汽包水位 (`h_drum`)。蒸汽流量作为系统的外部扰动。

## 6) 仿真设置
*   **仿真时长**：`500 s`
*   **通讯间隔/步长**：`0.01 s`
*   **求解器**：`dassl`
*   **容差**：`1e-06`

**结果分析**：仿真结果表明，PI 控制器能有效应对蒸汽流量的阶跃扰动。在扰动发生后，系统动态调整给水阀门，使实际水汽比例 (`ratio_actual`) 迅速回归并稳定在设定值 `1.0`，稳态误差极小。汽包水位 (`h_drum`) 在整个过程中变化平缓，体现了控制系统的有效性。

## 8) 运行与复现方法
1.  **环境准备**：确保安装有支持 Modelica 的仿真环境（如 Dymola，OpenModelica）。
2.  **加载模型**：在仿真工具中打开模型文件 `./model/BoilerFeedwaterRatio.mo`。
3.  **检查参数**：确认模型参数与本文档第3节“参数表”一致。无需额外覆盖。
4.  **设置仿真**：配置仿真参数：停止时间 `500 s`，输出间隔/步长 `0.01 s`，求解器选择 `dassl`，容差 `1e-06`。
5.  **执行仿真**：运行瞬态仿真。
6.  **查看结果**：绘制或导出第7节中提到的关键变量，以复现所述动态过程。
