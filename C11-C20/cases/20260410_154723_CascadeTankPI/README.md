# 串级PI控制双水箱液位系统仿真模型 (CascadeTankPI)

## 1) 模型目的与范围
本模型旨在仿真并控制一个串联双水箱液位系统。通过建立重力驱动出流与质量守恒的流体动力学机制，模拟上下两个水箱的液位变化过程。模型的核心目标是应用串级PI控制策略，克服单回路控制的滞后性，使下水箱的液位快速、平稳地跟踪设定值，并抑制系统内部扰动。

**物理机制涵盖：**
- 质量守恒
- 重力驱动出流
- 串联水箱流体动力学

## 2) 组件清单
本模型使用了以下 Modelica 标准库 (MSL) 组件：

| 组件路径 | 用途说明 |
| :--- | :--- |
| `Modelica.Blocks.Sources.Constant` | 生成下水箱液位设定值常数 |
| `Modelica.Blocks.Sources.RealExpression` | 将物理变量（h1, h2）转换为控制模块信号 |
| `Modelica.Blocks.Math.Feedback` | 计算设定值与实际值的反馈偏差 |
| `Modelica.Blocks.Continuous.PI` | PI 控制器（分别用于外环和内环） |
| `Modelica.Blocks.Nonlinear.Limiter` | 限制阀门开度信号在 [0, 1] 之间 |

## 5) 控制逻辑说明
本模型采用**串级PI控制策略**，包含外环（主环）和内环（副环）两个控制回路：

1. **外环控制（主回路）**：以下水箱液位 `h2` 为被控量，以下水箱设定液位 `h2_ref` 为目标。外环PI控制器根据液位偏差 `h2_error = h2_ref - h2` 进行计算，其输出 `u_outer` 作为内环的设定值（即上水箱的目标液位）。
2. **内环控制（副回路）**：以上水箱液位 `h1` 为被控量，以外环控制器的输出 `u_outer` 为目标。内环PI控制器根据偏差 `u_outer - h1` 进行计算，其输出 `u_inner` 代表阀门开度指令。
3. **执行与限幅**：内环输出指令经过限幅器（`Limiter`），确保阀门开度在 `[0, 1]` 范围内。实际进入上水箱的流量为 `Q_in = valve_limiter.y * Q_in_max`。
4. **物理过程**：流体流入上水箱后，经重力作用从上水箱流至下水箱（`Q_12`），再从下水箱流出（`Q_out`），流量大小受相应水箱液位及流量系数决定。

## 8) 运行与复现方法
1. **模型文件**：请确保模型文件位于相对路径 `./model/CascadeTankPI.mo`。
2. **环境要求**：需要支持 Modelica 语言的仿真环境（如 Dymola, OpenModelica, OMSimulator 等）。
3. **加载与仿真**：
   - 在仿真环境中加载 `./model/CascadeTankPI.mo` 文件。
   - 选择模型 `CascadeTankPI`。
   - 按照上述【第6节】配置仿真参数（StopTime=600, StepSize=0.01, Solver=dassl, Tolerance=1e-06）。
   - 运行仿真并绘制 `h1`, `h2`, `h2_error`, `Q_in`, `Q_12`, `Q_out`, `u_outer`, `u_inner` 等目标变量曲线以复现结果。
