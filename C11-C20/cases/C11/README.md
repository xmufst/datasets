# ThermostatWaterTank 模型文档

## 1) 模型目的与范围

本模型 `ThermostatWaterTank` 实现了一个恒温热水箱的闭环控制系统。其核心目的是模拟一个采用滞回（继电器）控制的电加热水箱的动态热行为。模型通过比较水箱温度与设定温度，控制加热器的启停，使水温维持在设定范围内。

**物理机制**：
*   集总热容模型
*   与环境的热耗散（与温差成正比）
*   基于滞回带的继电器控制

**模型范围**：适用于分析简单温控系统的动态特性、能耗以及控制逻辑的稳定性，例如家用热水器或小型工业加热罐的初步仿真。

## 2) 组件清单（MSL 组件）

模型仿真从以下初始状态开始：
*   **水温 (`T`)**：`25.0 °C`
*   **加热器状态 (`hysteresis.y`)**：`false` (关闭)

## 5) 控制逻辑说明

控制系统采用典型的滞回（双位）控制策略，逻辑流程如下：
1.  **误差计算**：连续计算目标温度 `T_set` 与实测水温 `T` 的偏差 `T_error`。
2.  **滞回判断**：偏差 `T_error` 送入滞回继电器 `Hysteresis`。
    *   当 `T_error > deadband/2` (即 `T < T_set - deadband/2`) 时，继电器输出 `true`，开启加热器。
    *   当 `T_error < -deadband/2` (即 `T > T_set + deadband/2`) 时，继电器输出 `false`，关闭加热器。
    *   在 `-deadband/2 <= T_error <= deadband/2` 区间内，继电器保持上一个状态。
3.  **功率输出**：继电器的布尔输出通过 `BooleanToReal` 块转换为物理加热功率 `Q_in`（开启时为 `Q_heater`，关闭时为 `0`）。

此逻辑确保了水温在 `T_set ± deadband/2` 的范围内波动，避免了加热器的频繁动作。
