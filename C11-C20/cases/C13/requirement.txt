系统仿真需求书：直流电机状态反馈控制模型（DCMotorStateFeedback）
1. 系统组成和拓扑描述
模拟一个利用 Modelica 标准库（MSL）多领域组件实现的直流电机全状态反馈转速跟踪控制系统。
系统分为“物理对象（电机本体）”和“控制回路”两部分，主体框架必须面向对象装配，必须通过信号连线（connect）或端口赋值实现系统闭环。
物理对象（直流电机）：
电机由电气和机械两个子系统耦合而成，可使用基础物理方程描述。
电气动态：电枢电感乘以电枢电流的变化率，等于施加的电枢电压减去电阻上的压降，再减去电机的反电动势。其中，反电动势等于反电动势常数乘以转子转速。
机械动态：转子的转动惯量乘以转速的变化率，等于电机产生的电磁转矩减去粘性摩擦转矩，再减去外部负载转矩。其中，电磁转矩等于转矩常数乘以电枢电流；粘性摩擦转矩等于摩擦系数乘以转速。
控制回路与扰动拓扑（必须检索并实例化 MSL 标准控制库中的对应功能组件）：
• 参考转速模块：需要一个阶跃信号源模块，在 t=0 时刻起输出给定的目标参考转速。
• 负载扰动模块：需要另一个阶跃信号源模块，在指定的负载施加时刻（t_load）起，输出一个阶跃的外部负载转矩，连接或赋值给电机本体的外部负载输入。
• 前馈与反馈增益模块：需要分别实例化三个独立的数学比例增益模块。第一个用于放大参考转速信号（前馈增益）；第二个用于放大实际电枢电流信号（状态反馈1）；第三个用于放大实际转子转速信号（状态反馈2）。
• 控制合成模块：需要一个多输入的数学求和/相减模块。该模块负责合成最终的电枢电压控制信号：将经过增益放大的参考转速（作为正项）减去经过增益放大的电枢电流，再减去经过增益放大的转速。合成后的连续信号作为电机的电枢电压。
初始参数和条件禁止硬编码(写死)。
2. 参数名称和数值
电枢电阻：Ra = 1.0 Ω
电枢电感：La = 0.01 H
反电动势常数：Ke = 0.1 V·s/rad
转矩常数：Kt = 0.1 N·m/A
转动惯量：J = 0.01 kg·m²
粘性摩擦系数：B_fric = 0.001 N·m·s/rad
参考转速：omega_ref = 100.0 rad/s
电流反馈增益：K1 = 0.5
转速反馈增益：K2 = 0.5
参考前馈增益：Kr = 0.62
负载转矩阶跃幅值：T_load_step = 0.05 N·m
负载施加时刻：t_load = 5.0 s
3. 初始条件
初始电枢电流：ia 在 t=0 时刻严格等于 0
初始转速：omega 在 t=0 时刻严格等于 0
4. 目标变量名称
omega：转子转速
ia：电枢电流
Va：施加的电枢电压（等于控制合成模块的输出）
Vemf：反电动势
T_motor：电机电磁转矩
T_load：外部负载转矩（对应负载扰动模块的输出）
omega_error：转速跟踪误差（定义为 omega_ref - omega）
u_control：控制信号（数值上恒等于 Va，作为辅助别名输出）
5. 仿真参数（强制，不可更改）
仿真总时长：stop_time = 10 秒
输出步长：step_size = 0.01 秒
求解器：solver_method = dassl
容差：tolerance = 1e-6
