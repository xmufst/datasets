# SpringMassGainScheduling 模型文档

## 1) 模型目的与范围
本模型 `SpringMassGainScheduling` 是一个非线性弹簧质量增益调度控制系统的仿真示例。模型核心是一个带有立方非线性刚度的弹簧质量阻尼系统，其控制器采用增益调度策略：比例增益 `Kp` 根据质量块位移 `x` 的绝对值大小，在三个不同的区域（低、中、高）间切换，以实现更好的跟踪性能和控制效果。模型旨在演示针对非线性系统的增益调度控制策略的设计与仿真。

**主要范围**：模拟一个受控的单自由度机械系统对阶跃参考信号的动态响应，研究非线性弹簧特性和变增益控制对系统性能（如跟踪误差、超调、稳定时间）的影响。

## 2) 组件清单（Modelica 标准库组件）
本模型主要基于方程构建，使用了以下 Modelica 标准库 (MSL) 组件：

*   `Modelica.Blocks.Sources.Step`：用于生成阶跃形式的参考位置信号 `x_ref_signal`。

## 5) 控制逻辑说明
模型实现了一个基于位移的分区增益调度 PD（比例-微分）控制器。

1.  **误差计算**：控制目标为跟踪阶跃参考信号 `x_ref_signal`。跟踪误差定义为 `x_error = x_ref_signal - x`。

2.  **增益调度**：当前比例增益 `Kp_current` 根据质量块位移 `x` 的绝对值实时切换：
    *   **低区**：当 `abs(x) < zone1` (0.5 m) 时，`Kp_current = Kp_low` (30 N/m)。
    *   **中区**：当 `zone1 ≤ abs(x) < zone2` (0.5 ~ 1.5 m) 时，`Kp_current = Kp_mid` (50 N/m)。
    *   **高区**：当 `abs(x) ≥ zone2` (1.5 m) 时，`Kp_current = Kp_high` (80 N/m)。
    *   微分增益 `Kd` (5 N·s/m) 在整个过程中保持不变。

3.  **控制律**：控制力 `F_control` 由 PD 控制律产生：
    ```
    F_control = Kp_current * x_error - Kd * v
    ```

4.  **被控对象**：系统动力学由牛顿第二定律描述，受力包括控制力 `F_control`、非线性弹簧力 `F_spring` 和粘性阻尼力 `c * v`。
    *   非线性弹簧力：`F_spring = k_base * x + k_nl * x^3`

## 8) 运行与复现方法
1.  **准备模型文件**：将提供的 Modelica 代码保存为 `./model/SpringMassGainScheduling.mo`。
2.  **加载模型**：在 OpenModelica OMEdit 或兼容的 Modelica 仿真环境中，加载此模型文件。
3.  **检查设置**：确保模型注释中的仿真设置（`StopTime=20, Interval=0.01`）已生效，或手动设置相同的停止时间和输出间隔。
4.  **仿真运行**：执行仿真。模型无外部参数需要覆盖，直接运行即可。
5.  **查看结果**：仿真结束后，可绘制 `x`, `x_ref_signal`, `x_error`, `v`, `F_control`, `Kp_current` 等关键变量以分析系统动态和控制行为。
